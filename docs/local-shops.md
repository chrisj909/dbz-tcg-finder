# Birmingham, AL — Local Sourcing Guide

Real, currently-operating card / game / hobby / comic shops + collectible venues in the Birmingham metro for sourcing **DBZ sealed product (Score / Panini / Bandai)** and **DBZ merch**. Companion to `docs/sources.md` (online sources) and the Phase 3 local-sources work.

Researched 2026-06. Addresses/handles verified against multiple listings where possible; **fields are omitted when unverifiable** — do not assume blanks mean "none." Verify before relying. Re-check `dbs-cardgame.com` store list periodically (it's the authoritative signal for DBSCG stock).

## How to use this

- **Watch list** = shops with a website / social we can monitor programmatically (Phase 3+: poll store sites, TCGplayer Pro storefronts, FB/IG feeds for new sealed DBZ/DBSCG listings). These feed the scanner; they are **not** browser-login sources like FB Marketplace — most are public HTML / public TCGplayer Pro pages, so they can run anywhere (cloud or local). Respect each site's robots.txt + polite delays per the `CLAUDE.md` hard rules.
- **Call / visit list** = shops with little/no online inventory. **Chris action, manual.** Phone or drop in to ask about sealed DBZ (all eras) + DBZ figures/merch, and ask to be texted when product walks in.
- **Shows / cons** = recurring in-person events where sealed vintage and merch surface; check the dated aggregators before each.
- Per hard rules: **no auto-buy, no auto-message** — anything outbound is a draft for Chris.

The strongest DBSCG signal: **Pinnacle Cards and Games** (Vestavia Hills) is on the *official* Dragon Ball Super Card Game store list, so it actively stocks/orders DBSCG. Treat its site + IG as a priority feed.
